🔧 Sobre o Projeto — ETL com Pentaho
Pipeline completo de Engenharia de Dados com Pentaho PDI/Kettle — extração, transformação, merge de tabelas e carga em banco de dados relacional.




⚙️ Fluxo do Pipeline ETL

<img src="https://img.shields.io/badge/1-Extract-378ADD?style=flat-square&logoColor=white" height="22" /> **Extract** — Leitura de múltiplas fontes: CSV, Excel e banco de dados via `CSV File Input` / `Table Input`


<img src="https://img.shields.io/badge/2-Transform-1D9E75?style=flat-square&logoColor=white" height="22" /> **Transform** — Limpeza, validação e enriquecimento dos dados com `Filter Rows` / `String Operations`


<img src="https://img.shields.io/badge/3-Merge-FF6B00?style=flat-square&logoColor=white" height="22" /> **Merge** — Junção e comparação de tabelas com `Merge Rows (diff)` / `Join Rows`


<img src="https://img.shields.io/badge/4-Load-6E40C9?style=flat-square&logoColor=white" height="22" /> **Load** — Carga no banco de dados relacional via `Table Output` / `Insert-Update`
